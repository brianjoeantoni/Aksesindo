Successful Implementation of IAX Products on Amerika Latin cluster gateways in Green Lake City on August 31, 2022.

Client Summary:

The Latin Amerika Latin cluster is one of the clusters in Green Lake City which has more than 567 housing units. Where one housing unit can have more than one vehicle.

Implementation Scope

The Amerika Latin cluster implements an access control system on 4 gateway clusters using IA-X2 products and devices that use L-Intelligent Access software which stores thousands of card credentials access. The IAX access controller provides a dual security system with Access card validation abd cannot be used by more than one vehicles inside at the same time.
